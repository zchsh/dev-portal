# Button

The `Button` component should be used when [an HTML `<button>` element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button) is needed in a style according to the design system.

> ðŸš¨ This component does not render [an HTML `<a>` element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a). See related [`InlineLink`](/swingset/components/inlinelink) or [`StandaloneLink`](/swingset/components/standalonelink) for components that render `<a>` elements.

## Props

ðŸš¨ Guidance on icon sizes to pass into the `icon` prop:

- If `size` is being set to "small" or "medium", then size 16 icons are recommended.
- If `size` is being set to "large", then size 24 icons are recommended.

## Icon only

If the `icon` prop is provided and no prop `text` is provided, the component will throw an error if there is no accessible label given.

ðŸš¨ Guidance on accessible labels:

- `ariaLabel` and `ariaLabelledBy` cannot both be provided.
- `ariaDescribedBy` can be provided in addition to `ariaLabel` or `ariaLabelledBy`.
- A button's label should concisely express what will happen when the button is clicked. Examples:
  - "Copy to clipboard"
  - "Add to cart"
  - "Undo changes"

<LiveComponent>{`
<>
  <p>With <code>ariaLabel</code>, a screen reader should say "Copy, button" in this example.</p>
  <Button
    ariaLabel="Copy"
    icon={<SwingsetTestIcon name="copy" />}
    size="small"
  />
  <br />
  <p>With <code>ariaLabelledBy</code>, a screen reader should say "Add, button" in this example.</p>
  <p aria-hidden id="button-label" style={{ display: 'none', visibility: 'hidden'}}>Add</p>
  <Button
    ariaLabelledBy="button-label"
    icon={<SwingsetTestIcon name="plus" />}
    size="medium"
  />
  <br />
  <p>With <code>ariaDescribedBy</code>, a screen reader should say "Next, button" in this example.</p>
  <p aria-hidden id="button-description" style={{ display: 'none', visibility: 'hidden'}}>Next</p>
  <Button
    ariaDescribedBy="button-description"
    icon={<SwingsetTestIcon name="arrow-right" size={24} />}
    size="large"
  />
  <br />
</>
`}</LiveComponent>
